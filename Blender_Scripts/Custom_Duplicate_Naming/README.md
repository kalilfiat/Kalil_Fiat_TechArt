Blender Custom Duplicate Naming

A Blender addon that replaces the default duplicate naming system with a fully customizable workflow.

Instead of Blender’s automatic .001, .002, etc., this addon lets you define exactly how duplicated objects are named, giving you cleaner and more predictable results.

Features:

Custom naming for duplicated objects
Select separator: _, - or .
Counter types: numeric (1, 01, 001) or alphabetic (A, B, C...)
Configurable digit padding
Optional override of Shift + D duplication
Simple and fast UI panel
Toggle on/off at any time

Why use this:
Blender’s default naming system can quickly become messy in complex scenes. This addon gives you full control over naming, making your workflow cleaner, faster, and more production-friendly.

Installation:

Download the .py file
Open Blender
Go to Edit > Preferences > Add-ons
Click Install
Select the file
Enable the addon

Usage:

Open the sidebar with N
Find the addon panel
Choose your separator, counter type, and digit format
Optionally enable Shift + D override
Duplicate objects as usual

Example outputs:
Object_01
Object_02
Object_03

Object-001
Object-002
Object-003

Object-A
Object-B
Object-C

Notes:

Shift + D override replaces Blender’s default duplication behavior
Can be enabled or disabled anytime
Lightweight and non-destructive
