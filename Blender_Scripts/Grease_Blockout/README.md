Grease Blockout

Grease Blockout is a Blender addon designed to quickly generate 3D blockout meshes from Grease Pencil drawings. It is focused on fast ideation, prototyping, and early-stage modeling.

FEATURES

Convert Grease Pencil drawings into 3D meshes in one click
Adjustable mesh thickness
Depth direction control (Centered, Forward, Backward)
Symmetry on X, Y, or Z axis
Optional automatic voxel remesh
Replace previous mesh from the same sketch
Random soft material colors
Snap active sketch to a configurable grid
Quick layer management (create, clear, reset)
Automatic collection organization
Incremental naming for meshes
Voxel remesh presets (Coarse, Medium, Fine)

WORKFLOW

Create or activate the drawing object
Draw a closed shape with Grease Pencil
Generate the mesh
Optionally refine with remesh, symmetry, or new sketches

INSTALLATION

Download the .py file
Open Blender
Go to Edit > Preferences > Add-ons
Click Install
Select the file
Enable the addon

Panel location:
View3D > Sidebar > Blockout

USAGE

DRAWING

Click "Create / Activate Draw Object"
Enter "Draw Mode"

MESH GENERATION

Draw a closed shape
Click "Generate Mesh"

MAIN SETTINGS

Thickness: Controls mesh depth
Direction: Extrusion direction
Symmetry: Mirror across axis
Voxel Size: Remesh resolution
Auto Remesh: Apply remesh automatically
Random Soft Color: Assign random material
Replace Same Sketch Mesh: Avoid duplicates
New Layer After Generate: Create new layer after mesh

TOOLS

New Blank Sketch Layer
Clear Current Sketch
Clear All Sketches
Snap Active Sketch To Grid
Apply Voxel Remesh

NOTES

Works best with closed shapes
Uses active layer and current frame
Meshes store source layer and frame
Geometry is organized automatically in a collection
